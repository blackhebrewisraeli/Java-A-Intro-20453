 # Java-A-Intro-20453

A collection of Java assignments completed as part of course 20453 – Introduction to Computer Science and Java A.

This repository consolidates several separate assignment repositories into one organized course portfolio.

## Repository Structure

```text
assignments/
├── 01-train/
├── 02-point/
├── 03-date/
├── 04-city/
└── 05-matrix/
```

## Assignments

| Folder | Topic |
|---|---|
| `01-train` | Basic Java program structure, input, arithmetic, and conditionals |
| `02-point` | Object-oriented programming and 2D geometry |
| `03-date` | Date representation, validation, comparison, and calendar logic |
| `04-city` | Object composition using `City`, `Point`, and `Date` classes |
| `05-matrix` | Matrix operations and grayscale image representation |

## Technologies

- Java
- Object-Oriented Programming
- Encapsulation
- Input validation
- Basic algorithms
- Matrix manipulation

## Notes

These assignments were originally developed as separate repositories and were later consolidated into this repository for better organization and presentation.
